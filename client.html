Language files

The text strings (menu names, etc.) in the Windows GUI client are stored in a file called language.ini. The release uses American English. Other languages are available here (thanks to Robi Buechler and other volunteers for this).

The command-line client

Command-line options:

-add_new_project
Enroll in a new project. You will be asked for the project URL and an account ID.
-help
Show client options.
-version
Show client version.

To remove a project: quit the client. Then delete the file "account_PROJECT-URL.xml" where PROJECT-URL is the project's URL.
